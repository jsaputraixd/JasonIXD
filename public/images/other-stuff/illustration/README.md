# Illustration

Add illustration files here (JPG, PNG, or WebP), then register each one in `data/otherStuff.js`:

```js
// data/otherStuff.js → categories → illustration → items
{
  id: "piece-name",
  src: "/images/other-stuff/illustration/piece-name.jpg",
  alt: "What the viewer should know",
  caption: "Optional title under the thumbnail",
}
```
