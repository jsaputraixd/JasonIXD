# Motion

Use this folder for motion stills, GIFs, or short videos. Add entries under `motion` → `items` in `data/otherStuff.js`.

**GitHub limit:** files must be under **100MB**. `JasonSaputra_TitleSeq3_fa25.mp4` (~175MB) stays local (see root `.gitignore`). Compress to WebM/MP4 under 100MB, or use [Git LFS](https://git-lfs.github.com), then register in `data/otherStuff.js` with `type: "video"`.
