# Other stuff — media folders

Drop images (JPG, PNG, WebP) into a category folder, then register each file in `data/otherStuff.js`.

| Folder | Category |
|--------|----------|
| `photography/` | Photography |
| `illustration/` | Illustration |
| `drawing/` | Drawing |
| `sculptures/` | Sculptures |
| `motion/` | Motion (video or still frames) |

Example entry in `data/otherStuff.js`:

```js
{
  id: "my-piece-01",
  src: "/images/other-stuff/illustration/my-piece-01.jpg",
  alt: "Short description",
  caption: "Optional label",
}
```
